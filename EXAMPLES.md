# 脱敏案例集

以下案例为模拟/脱敏处理，仅用于展示 Skill 输出效果。

## 案例一：企业增长诊断

### 输入信息

**行业**：教育培训
**规模**：10-50人
**主诉痛点**：获客难、复制不了

### 关键诊断结论

```markdown
【快速扫描结果】

■ 6维评分
- 获客能力：2分 ← 核心瓶颈
- 盈利效率：4分
- 复购与推荐：3分
- 复制能力：1分 ← 次要瓶颈
- 组织能力：2分
- 战略清晰：3分

■ 核心瓶颈
- 最严重：获客能力 — 依赖创始人个人资源，无稳定渠道
- 次要：复制能力 — 严重依赖创始人，无法标准化

■ 建议深入模块
- 主攻：模块A（获客渠道）
- 辅攻：模块C（复制能力）+ 模块H（SOP建设）

【深度诊断发现】

■ 模块A：获客
- 关键发现：获客80%依赖创始人朋友圈和老客户转介绍，线上渠道几乎为零
- 根因分析：创始人认知里"做教育要靠口碑，不需要投广告"，拒绝尝试线上获客

■ 模块C：复制
- 关键发现：课程内容、教学流程、服务标准都在创始人脑子里，没有成文的SOP
- 根因分析：创始人认为"我这样做效果好，别人学不来"，低估了标准化的价值

【增长杠杆 + 改进路径】

■ 短期（1-3个月）
1. 梳理1-2个核心获客渠道的 SOP，降低对创始人个人资源的依赖
2. 建立最简易的课程交付标准文档，让新老师能快速上手

■ 中期（3-6个月）
1. 测试1-2个线上获客渠道（抖音/小红书），验证ROI
2. 完成核心业务流程的标准化，输出3-5个关键SOP

■ 长期（6-12个月）
1. 建立可复制的单店模型，为扩张做准备
2. 搭建团队，培养2-3个能够独当一面的核心员工
```

## 案例二：绩效沟通双向诊断

### 输入对话

```markdown
【上级】：小王，这个月你的业绩又没达标，怎么回事？
【员工】：嗯...最近家里事有点多，我也在尽力推进，就是市场环境不太好，同事那边配合也有点慢。
【上级】：家里的事我理解，但业绩是硬指标。别人也能完成，你怎么就完成不了？是不是得反思一下自己的问题？
【员工】：好的领导，我知道了，我会再想想办法的。
【上级】：那就这样吧，下个月必须达标，不然考核会很被动。
【员工】：嗯...好的。
【上级】：还有，这个月迟到3次，怎么回事？
【员工】：有时候送孩子上兴趣班，偶尔会晚一点。
【上级】：制度就是制度，迟到就是迟到，没有任何借口。
【员工】：...（沉默）
```

### 诊断输出（节选）

```markdown
【对话概览】
- 总字数：286字
- 对话轮次：8轮
- 说话人身份：上级-员工

【潜台词解码】

员工："家里事有点多，我也在尽力推进"
→ 真实意图：隐性回避，用外部原因（家里事、市场环境、同事配合）掩盖真实的绩效压力或个人困难
→ 原文证据："家里事"、"市场环境不太好"、"同事那边配合也有点慢"（三连外部归因）

员工："好的领导，我知道了"
→ 真实意图：口服心不服的敷衍式回应，并非真正的理解和承诺
→ 原文证据：重复"好的"，语气被动，回应长度递减

员工：[沉默]
→ 真实意图：防御性沉默，已关闭真实表达通道
→ 原文证据：上级强调"制度就是制度，没有任何借口"后员工直接沉默

【上级沟通行为评估】

⚠️ 维度2：给下属留面子的表达空间逻辑
- 不当行为：直接戳破员工的托词，关闭员工真实表达通道
- 原文证据："是不是得反思一下自己的问题？"——审讯式追问
- 对员工的直接影响：员工后续回复明显变短，进入"敷衍配合"模式

⚠️ 维度5：公开表扬私下纠错的反馈逻辑
- 不当行为：在公开场合（即使只有两人）提及迟到等负面小事，强化"被监视"感
- 原文证据："迟到就是迟到，没有任何借口"——对事也伤人的表达方式
- 对员工的直接影响：员工从"解释"直接切换为"沉默"，对话彻底进入对抗状态

【核心问题定性】

基于吉尔伯特行为工程模型：
- 员工自身能力问题：30%
- 员工自身动力问题：30%
- 上级管理行为导致的问题：40% ← 主因

【员工流失风险】

风险等级：高

触发原因：
1. 保健因素不满：公平性感知低（"别人也能完成，你怎么就完成不了"暗示差异化管理）
2. 尊重需求未满足：被当众质疑和追问，面子受损
3. 上级未提供支持：只压责任，不给资源和方法

【上级自我优化建议】

1. 下次绩效沟通，先用"我看到你最近...我很好奇..."开场，而非直接问责
2. 把"别人也能完成，你怎么就完成不了"换成"我们来一起看看卡在哪里了"
3. 迟到等私人话题放到最后单独说，不要在绩效谈话中作为"证据"提起
```

## 使用说明

以上案例为脱敏处理后的模拟数据，实际使用中：

1. **企业增长诊断**：提供的信息越详细，诊断越精准
2. **绩效沟通诊断**：对话文本需获得当事人知情同意

如需了解完整输出格式，请参考各 Skill 的 `SKILL.md` 文件。
