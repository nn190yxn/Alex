# 企业增长诊断顾问

企业增长诊断框架。通过行业诊断→创始人能力评估→快速扫描→深度诊断→方案输出的完整流程，识别中小企业增长瓶颈，输出可落地的增长方案。

## 触发词

- `/增长诊断`
- `/企业诊断`
- `帮我诊断企业增长`
- `企业增长瓶颈`
- `诊断一下我的生意`

## 功能模块

| 模块 | 说明 |
|------|------|
| 阶段0：行业诊断 | 8问快速定位行业特征与核心痛点 |
| 模块F：创始人诊断 | 评估6项核心能力（直接版/间接版可选） |
| 差异化定位 | 5步法找到可执行差异化路径 |
| 阶段1：快速扫描 | 6维度评分快速定位问题 |
| 阶段2：深度诊断 | 按规模分层深入挖掘根因 |
| 阶段3：综合报告 | 输出结构化诊断报告+增长方案 |

## 适用规模

- 1-10人（基础验证期）
- 10-50人（模式打磨期）
- 50-200人（效率优化期）
- 200人+（战略升级期）

## 诊断时长预估

| 规模 | 总计 |
|------|------|
| 1-10人 | ~55min |
| 10-50人 | ~95min |
| 50-200人 | ~125min |
| 200人+ | ~165min |

## 使用方式

1. 发送触发词启动诊断
2. 按提示逐一回答问题
3. 获得完整诊断报告
